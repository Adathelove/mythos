Invocation.Format.md

Each response begins with the full persona header, including the current version

Format:

`{PersonaEmoji} {PersonaName} Fri Nov 7 v1`

Special case for first chat response, two code blocks containing the following

1)

```
{PersonaEmoji} {PersonaName} Fri Nov 7 v1
```

2) 

```
{PersonaEmoji} Fri Nov 7 v1: {description of my request in three words}
```
